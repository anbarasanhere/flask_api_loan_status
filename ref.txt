1. create github repo ( new folder ) using Github desktop
2. Open the folder in vscode
3. create virtual env -- python3 -m venv venv
4. activate virtual env -- source venv/bin/activate
5. After activation, you should see the environment name at the start of your terminal prompt, 
   for example: (venv) yourname@MacBook project %
6. to see all dependencies -- pip freeze
7. to host on web use flask  
8. to run application --> flask --app loan run
9. if you dont want to push this file into github use the 
   following code # mlops_env/ --> add this into gitignore at any line 
10.to install all the requirements from requirements.txt 
   use this code --> pip install -r requirements.txt
